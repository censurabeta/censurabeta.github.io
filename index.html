If that isn't enough, here's a few more reasons why you should use Censura in your project:

Censura is written in blazing fast C++ - so fast, in fact, it can find a misspelled term in as little as 2 milliseconds or less!
Censura is a gRPC microservice that can be integrated quickly into any project written in a language that supports gRPC (C, C++, C#, Go, Java, Node.js, PHP, Python, and Ruby to name a few).
Censura is agile and scalable - it can easily be containerized and scaled on several popular schedulers and cloud services.
Censura's no-nonsense API allows you to fully integrate your application in minutes. Just send a blacklist, whitelist, and phrase, and Censura will tell you if it finds any blacklisted terms.
